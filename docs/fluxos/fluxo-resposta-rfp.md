# Fluxo — Resposta a RFP

## Objetivo de negócio
Responder edital formal com matriz de atendimento integral aos requisitos.

## Sequência de Skills

1. `_core/configurador-empresa`
2. `produtividade/extrator-dados` — extrair requisitos do edital (PDF/Word).
3. `conhecimento/classificador-documentos` — classificar anexos do edital.
4. `comercial/escopo-tecnico` — detalhar escopo.
5. `financeiro/analise-custos` + `comercial/orcamento`
6. `comercial/resposta-rfp` — matriz requisito→resposta.
7. `comercial/proposta-comercial` — compor documento final.
8. `administrativo/oficio-memorando` — ofício de entrega (se exigido).

## Saída final

- Matriz de atendimento completa
- Documentos técnicos anexos
- Proposta comercial formal

## Quando usar
- Editais governamentais
- Compras corporativas estruturadas
- Processos multinacionais

## Pré-requisitos
- Documento do edital em formato tratável
- Equipe técnica disponível para validar respostas
