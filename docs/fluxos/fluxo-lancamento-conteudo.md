# Fluxo — Lançamento de Conteúdo

## Objetivo de negócio
Lançar conteúdo multicanal com alinhamento editorial.

## Sequência de Skills

1. `marketing/artigo-blog` — artigo-pilar.
2. `marketing/conteudo-linkedin` — posts de divulgação.
3. `marketing/conteudo-site` — landing page.
4. `marketing/campanha-email` — sequência de e-mails.
5. `marketing/case-de-sucesso` — case correlato (se houver).
6. `produtividade/gerador-apresentacao` — slides de apresentação interna.
7. `_core/organizador-informacao` — briefing consolidado.
