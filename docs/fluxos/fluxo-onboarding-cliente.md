# Fluxo — Onboarding de Cliente

## Objetivo de negócio
Iniciar trabalho com cliente recém-contratado, garantindo alinhamento e governança.

## Sequência de Skills

1. `comercial/follow-up` — mensagem de boas-vindas.
2. `administrativo/email-corporativo` — agendar reunião de kick-off.
3. `administrativo/ata-reuniao` — documentar kick-off.
4. `gestao/plano-acao` — plano de trabalho do primeiro mês.
5. `gestao/cronograma-projeto` — cronograma macro.
6. `gestao/gestao-riscos` — riscos iniciais do projeto.
7. `documentacao/pop-sop` ou `documentacao/checklist` — rotinas aplicáveis ao cliente.
8. `financeiro/prestacao-contas` — template de prestação de contas (se aplicável).

## Saída final

- Mensagem de boas-vindas enviada
- Kick-off documentado
- Plano de trabalho + cronograma + riscos
- Templates prontos para uso
