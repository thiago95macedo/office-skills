# Fluxo — Análise Financeira de Proposta

## Objetivo de negócio
Avaliar viabilidade financeira de novo contrato/investimento antes de submeter à diretoria.

## Sequência de Skills

1. `comercial/escopo-tecnico` — definir entregas.
2. `financeiro/analise-custos` — decompor custos.
3. `financeiro/justificativa-financeira` — ROI, payback, riscos.
4. `financeiro/orcamento-empresarial` — impacto no orçamento anual.
5. `gestao/gestao-riscos` — riscos financeiros.
6. `gestao/dashboard-executivo` — resumo executivo para decisão.

## Saída final

- Decomposição de custos
- Justificativa estruturada
- Impacto orçamentário
- Riscos financeiros
- Dashboard de decisão
