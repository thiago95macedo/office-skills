# Fluxo — Auditoria Documental

## Objetivo de negócio
Revisar acervo de documentos com classificação, lacunas e recomendações.

## Sequência de Skills

1. `conhecimento/classificador-documentos` — classificar lote.
2. `produtividade/resumo-executivo` — resumir achados.
3. `suporte/base-conhecimento` — propor estrutura organizacional.
4. `conhecimento/mapa-conhecimento` — identificar lacunas.
5. `gestao/plano-acao` — plano de regularização.
6. `administrativo/ata-reuniao` — documentar revisão.

## Saída final

- Classificação completa
- Resumo executivo
- Estrutura proposta
- Mapa de conhecimento
- Plano de regularização
- Ata da revisão
