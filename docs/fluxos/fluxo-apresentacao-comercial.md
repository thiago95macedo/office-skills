# Fluxo — Apresentação Comercial

## Objetivo de negócio
Construir pitch/dump para cliente potencial ou conselho.

## Sequência de Skills

1. `_core/configurador-empresa`
2. `comercial/one-pager` — definir mensagem central.
3. `comercial/escopo-tecnico` — detalhar oferta.
4. `marketing/case-de-sucesso` — preparar 2-3 cases de evidência.
5. `produtividade/gerador-apresentacao` — estruturar slides.
6. `comercial/tratamento-objeções` — preparar respostas para Q&A.

## Saída final

- Apresentação estruturada
- Cases selecionados
- FAQ de Q&A

## Quando usar
- Pitches para investidor
- Apresentações para cliente
- Apresentações para conselho
