# Fluxos Compostos

Receitas de uso que combinam várias Skills da biblioteca Office Skills para entregar resultados completos de negócio.

Cada fluxo descreve:
- Objetivo de negócio
- Sequência de Skills
- Entradas e saídas intermediárias
- Saída final
- Quando usar
- Pré-requisitos

| Fluxo | Quando usar |
|-------|-------------|
| [Proposta Comercial Completa](fluxo-proposta-completa.md) | Responder RFP ou pedido formal de proposta |
| [Resposta a RFP](fluxo-resposta-rfp.md) | Editais com requisitos pontuais |
| [Apresentação Comercial](fluxo-apresentacao-comercial.md) | Pitches para cliente |
| [Onboarding de Cliente](fluxo-onboarding-cliente.md) | Iniciar trabalho após fechamento |
| [Onboarding de Colaborador](fluxo-onboarding-colaborador.md) | Integrar novo colaborador |
| [Atendimento e Triagem Omnichannel](fluxo-atendimento-omnichannel.md) | Operar caixa multi-canal |
| [Análise Financeira de Proposta](fluxo-analise-financeira.md) | Avaliar viabilidade de novo contrato |
| [Fechamento Mensal](fluxo-fechamento-mensal.md) | Rotina de fechamento |
| [Auditoria Documental](fluxo-auditoria-documental.md) | Revisão periódica de documentos |
| [Lançamento de Conteúdo](fluxo-lancamento-conteudo.md) | Sequência editorial multicanal |
