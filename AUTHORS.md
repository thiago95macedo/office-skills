## Como contribuir

Veja [`CONTRIBUTING.md`](CONTRIBUTING.md) para o processo completo.

Para entrar em contato sobre:

- **Dúvidas sobre uso** — abra uma issue no repositório.
- **Sugestões de novas Skills** — abra uma issue com a tag `enhancement`.
- **Bugs** — abra uma issue com a tag `bug`.
- **Segurança** — envie e-mail direto para `contato@thiagomacedo.com.br`.
- **Parcerias e uso comercial** — envie e-mail direto para `contato@thiagomacedo.com.br`.
