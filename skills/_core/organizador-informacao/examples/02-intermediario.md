# Exemplo intermediário — Notas de reunião

## Entrada
Notas soltas de reunião (5 participantes, 1h).

## Saída esperada
Estrutura:
1. Participantes
2. Pauta
3. Decisões
4. Ações (com responsável e prazo)
5. Próxima reunião

Cada item das notas é mapeado para a seção correspondente; redundâncias eliminadas.
