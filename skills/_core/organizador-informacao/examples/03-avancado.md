# Exemplo avançado — Compilar relatório trimestral

## Entrada
Planilhas + e-mails + atas de reunião.

## Saída esperada
Documento único com:
- Sumário executivo
- Indicadores-chave (com dados preservados)
- Ações em andamento
- Riscos
- Próximos passos

Citações de fonte em parênteses; nenhuma informação é inventada.
