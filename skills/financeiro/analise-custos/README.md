# Análise de Custos

Skill financeira essencial para decompor custos e sugerir preço de venda.

## Quando usar
- Definição ou repactuação de preços.
- Avaliação de rentabilidade de contratos.
- Subsídio a propostas comerciais.

## Arquivos desta Skill
- `SKILL.md` — definição canônica
- `prompt.md` — prompt otimizado para o agente
- `examples/` — três cenários (básico, intermediário, avançado)
- `templates/` — entrada e saída esperadas
- `tests/` — casos de teste

## Uso rápido
Veja `examples/01-basico.md` para um exemplo aplicado.
