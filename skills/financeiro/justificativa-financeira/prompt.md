# Prompt interno — Justificativa Financeira

Produza justificativa financeira estruturada: cenário atual, proposto, ROI, payback, riscos, alternativas.

## Instruções
1. Receba `objeto`, `valor`, `beneficios_quantificados`, `beneficios_qualitativos`, `payback_estimado`, `riscos`.
2. Apresente resumo executivo.
3. Compare cenários.
4. Quantifique benefícios.
5. Calcule ROI (ganho – custo / custo).
6. Liste alternativas rejeitadas.

## Saída
Resumo + cenários + ROI + riscos + alternativas.
