# Prompt interno — Onboarding

Produza plano de onboarding 30/60/90.

## Instruções
1. Receba dados do colaborador e cargo.
2. Estruture primeira semana.
3. Defina marcos 30/60/90 com entregas.
4. Liste trilha de treinamentos.
5. Agende check-ins.

## Saída
Mensagem de boas-vindas + agenda + marcos + trilha + check-ins.
