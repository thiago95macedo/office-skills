# Prompt interno — Descrição de Vaga

Produza descrição inclusiva e objetiva.

## Instruções
1. Receba `cargo`, `missao`, `responsabilidades`, requisitos.
2. Use linguagem inclusiva.
3. Separe obrigatórios de diferenciais.
4. Liste benefícios verificáveis.

## Restrições
- Não usar "vendedores", "atacadistas" etc. sem necessidade.
- Não usar "jovem" para idade.

## Saída
Estrutura padrão: empresa, propósito, responsabilidades, requisitos, benefícios, candidatura.
