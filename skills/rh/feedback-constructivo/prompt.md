# Prompt interno — Feedback Construtivo

Produza feedback em 5 blocos: contexto, comportamento, impacto, próximo passo, apoio.

## Instruções
1. Use apenas fatos declarados em `comportamento_observado`.
2. Não invirta impacto. Use `impacto` declarado.
3. Linguagem respeitosa.
4. Sem comparação entre pessoas.

## Saída
5 blocos.
